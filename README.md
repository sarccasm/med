# Med Cycle Tracker

Med Cycle Tracker — це невеликий веб-застосунок, який допомагає не збиватися з курсу ліків,
особливо коли лікування складається з кількох етапів (сильні препарати → легкі → перерва).

## Функціонал

- Реєстрація / вхід користувача (JWT, захищені API).
- Створення курсів з кількома фазами у форматі `дні: опис` (наприклад:
  `7: Сильні снодійні`, `5: Легкі снодійні`, `3: Без ліків`).
- Автоматичний розрахунок:
  - поточної фази;
  - дня у фазі;
  - кількості днів до переходу на наступний етап.
- Панель **«Сьогодні»**: показує, що саме потрібно прийняти сьогодні.
- Відмітка **«сьогодні вже прийнято»** для кожного курсу (щоденник прийомів у MongoDB).
- Нотатки за день (самопочуття, побічні ефекти), прив’язані до курсу й дати.
- Щоденні нагадування через Notification API (час задається вручну).
- Архівування курсів (розділення на активні й архівні).
- Редагування курсу: назва, дата початку, тривалість та опис фаз.
- Акуратний темний інтерфейс, адаптивний для різних розмірів екрана.

## Стек технологій

- **Frontend:** React + Parcel, CSS без бібліотек.
- **Backend:** Node.js, Express, JWT.
- **База даних:** MongoDB (Mongoose).

```bash

cp .env.example .env   # при потребі змінити MONGO_URI і JWT_SECRET
npm install
npm run dev

Клієнт
cd client
npm install
npm start


Клієнт відкриється на http://localhost:3000.
